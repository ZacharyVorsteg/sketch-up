## Part 6: Understanding the Interface Philosophy

### Why SketchUp Is Different

Most 3D software is complex. SketchUp is **intuitive**.

**The Core Philosophy:**
1. **Push/Pull** - Make flat things 3D by pushing/pulling
2. **Inference** - SketchUp guesses what you want
3. **Sticky Geometry** - Things connect automatically
4. **Components** - Reusable objects (we'll learn later)

**Example of Inference:**
- Move your mouse near an edge
- See it snap? That's inference
- See the colored dots at endpoints? Inference!
- See the "On Edge" tooltip? Inference!

**ğŸ’¡ Pro Tip:** Trust the inference engine. It's helping you.
