**Problem:** Extension hoarding
- Slow startup
- Cluttered menus
- Conflicting extensions
- Confusion

**Solution:** Only install what you actually use

**Rule:** If you haven't used it in a month, uninstall it

## Part 8: Troubleshooting Extensions

### Extension Won't Install

**Check:**
- SketchUp version compatibility
- File permissions (Run as Admin)
- .rbz file not corrupted
- Extension Warehouse login

---

### Extension Doesn't Appear

**Try:**
1. Window â†’ Extension Manager
2. Find extension
3. Enable it
4. Restart SketchUp
5. Check Extensions menu

---

### Extension Causes Crash

**Fix:**
1. Disable extension
2. Check for updates
3. Check compatibility
4. Report bug to developer
5. Find alternative

---

### Extensions Conflict

**Symptoms:**
- Unexpected behavior
- Tools not working
- Crashes

**Fix:**
1. Disable all extensions
2. Enable one by one
3. Find conflicting pair
4. Keep only one
5. Or update both
